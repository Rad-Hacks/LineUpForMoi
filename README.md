## How to Start the App
1. Setup the mysql server:  
  `$ mysql -u root < schema.sql`  
  `$ mysql`  
  `mysql> use q;`  
2. Start the back end server  
   `$ npm start`  
3. Start the front end server  
   `$ cd client && npm start`
